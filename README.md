# Estudio Jurídico Dra. Aranza Herrera
Sitio web profesional de la Dra. Aranza Herrera - Rosario, Santa Fe.

## Estructura del Proyecto
- `index.html`: Página principal.
- `equipo.html`: Información sobre el equipo legal.
- `contacto.html`: Formulario de contacto y agendamiento.
- `redes.html`: Enlaces a redes sociales y contenido de TikTok.
- `areas/`: Secciones detalladas de cada área de práctica.
- `css/`: Estilos personalizados.
- `js/`: Lógica de interactividad y animaciones.
